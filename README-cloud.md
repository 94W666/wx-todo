# 微信云开发配置指南

## 1. 开通云开发
1. 登录[微信公众平台](https://mp.weixin.qq.com/)
2. 进入小程序管理后台
3. 左侧菜单选择「云开发」
4. 点击「开通」按钮，按照指引完成开通
5. 创建云环境（建议创建两个：一个测试环境，一个生产环境）

## 2. 获取环境ID
1. 在云开发控制台，点击「设置」
2. 在「环境设置」标签页中，找到「环境ID」
3. 复制环境ID（格式如：`dev-xxxxxx` 或 `prod-xxxxxx`）

## 3. 修改配置文件
1. 打开 `app.js` 文件
2. 找到 `wx.cloud.init` 部分
3. 将 `env` 参数替换为你的环境ID：
   ```javascript
   wx.cloud.init({
     env: 'your-env-id', // 替换为你的环境ID
     traceUser: true,
   })
   ```

## 4. 创建数据库集合
在云开发控制台中创建以下集合：
1. **teams** - 存储团队信息
   - 索引建议：`teamCode`（唯一索引）
2. **teamPunchRecords** - 存储团队打卡记录
   - 索引建议：`teamId`、`date`、`userOpenId`

## 5. 部署云函数
1. 在微信开发者工具中，右键点击 `cloudfunctions` 文件夹
2. 选择「上传并部署：所有文件」
3. 或者逐个上传每个云函数文件夹

## 6. 测试步骤
1. 编译小程序
2. 点击首页的「组队打卡」入口
3. 测试创建团队、加入团队功能
4. 测试打卡同步功能

## 7. 注意事项
1. 云环境配额：注意云函数的调用次数和数据库的读写次数
2. 权限设置：确保数据库集合的权限设置正确（建议开发阶段设置为「所有用户可读，仅创建者可写」）
3. 版本管理：生产环境建议使用独立的环境ID

## 8. 故障排查
1. **云函数调用失败**：检查云函数是否部署成功，环境ID是否正确
2. **数据库操作失败**：检查集合名称是否正确，权限设置是否允许
3. **网络错误**：检查网络连接，确保小程序基础库版本支持云开发
